Outras curiosidades:

Você pensava que o falcão possuía uma visão única? Pois saiba que este pequeno animal consegue ver ainda melhor!

O Stomatopoda consegue enxergar 12 cores primárias distintas. Acreditem, este pequenino e aparentemente insignificante ser vivo possui o sistema de visão mais complexo de todo o reino animal.
